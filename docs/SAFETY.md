# 安全说明

## 不要提交这些内容

以下内容只能保存在本机，不要提交到 GitHub：

- 闲鱼 Cookie
- 账号密码
- 大模型 API Key
- 数据库文件
- 发货卡密、网盘链接、兑换码
- 日志文件
- 浏览器登录状态目录

本仓库已经通过 `.gitignore` 忽略 `.env`、`data/`、`logs/`、`backups/`、`vendor/` 等目录，但仍建议提交前运行：

```bash
git status
```

确认没有敏感文件。

## 后台不要公网裸奔

默认端口绑定：

```text
127.0.0.1:9000
```

这表示只能本机访问。不要轻易改成：

```text
0.0.0.0:9000
```

如果需要远程访问，优先使用 Tailscale、ZeroTier 或 Cloudflare Tunnel，并开启强密码。

## 自动发货风险控制

建议只对低风险虚拟商品开启自动发货，并设置以下规则：

| 风险 | 建议 |
| --- | --- |
| 高金额订单 | 转人工确认 |
| 新买家第一单 | 转人工确认或延迟发货 |
| 同一买家频繁下单 | 暂停自动发货 |
| 商品规则未命中 | 不发货，记录日志 |
| AI 无法判断 | 不发货，转人工 |

## AI 回复边界

AI 回复建议作为兜底客服，不要授权它做这些事：

- 承诺退款、赔偿、补偿
- 引导站外交易
- 编造商品状态或发货状态
- 泄露后台规则、API Key、Cookie
- 自动处理争议或投诉

## 平台规则提醒

自动回复和自动发货可能触发平台风控。请控制频率，只处理自己的正常交易，不要做批量营销、采集、刷量、规避验证或骚扰用户等行为。
